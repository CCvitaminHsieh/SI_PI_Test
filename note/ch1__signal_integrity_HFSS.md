1. 信號完整性的基本物理是什麼
	- 為什麼高速訊號不是 "電壓" 而是 "電磁波"
	- 為什麼 trace 是傳輸線
	- 為什麼 via, connector, 封裝互連會造成反射、損耗、串擾
	- 為什麼要用 S參數 (S11、S21) 來描述互連
	- 為什麼 3D EM 模擬比 2D 模擬更準確
	HFSS 是用來解 Maxwell 方程式的，不是用來畫 PCB 的。
	把 via、bump、TSV 看成「電磁結構」，而不是「電路」。

1.1. 常用軟件介紹
	- HFSS：3D 全波 EM（最準確）
	- SIwave：封裝/PCB 的 SI/PI（大模型）

	- 其他工具（Sigrity、Clarity、CST）
	
	每個工具的定位與邊界，不要用錯工具。
	例如：
	- HFSS 用來做 TSV、bump、RDL
	- SIwave 用來做封裝 PDN
	- ADS 用來做 HBM channel
	這對你未來做封裝模擬非常重要。

1.4. 設置材料及屬性
	你會學到：
	- 設定金屬（Cu、Al）
	- 設定介質（FR4、Rogers、SiO₂、Si）
	- 設定損耗（tanδ、σ）
	這是你後面做封裝（ABF、Si interposer）必備的。

1.5. 設置解算參數
	你會學到：
	- mesh 設定

	- port 設定（wave port、lumped port）

1.6. 查看結果

1.7. 總結
	第一章不是要你做封裝級模型，而是：

# 📡 第 2 章真正的主題：傳輸線電磁特性 + HFSS 實作

## 1) **傳輸線的物理直覺（微帶線 / 帶狀線）**
每一小節都在回答同一個問題：

> **幾何（線寬、線厚、介質厚度、介電常數）如何影響傳輸線的電磁特性？**

你會看到：

- 線寬變大 → 阻抗變小  
- 線厚變大 → 阻抗變小  
- 介質變厚 → 阻抗變大  
- 介電常數變大 → 阻抗變小  
- 微帶線 vs 帶狀線 → 有效介電常數不同  

這些都是 **微波工程的基本規律**，但透過 HFSS 你會看到它們的「場分佈」與「S 參數」如何變化。

這是你後面做封裝互連（TSV、bump、RDL）的基礎直覺。

## 2) **HFSS 建模能力（工程建立）**

## 🧩 第 3 章真正的主題：過孔的電磁行為 + HFSS 建模技巧

過孔是高速設計中最複雜的無源元件之一，因為它同時具有：

- **電感（via barrel）**  
- **電容（anti-pad、reference plane）**  
- **stub（殘樁）**  
- **耦合（相鄰 via）**

第 3 章的每一節都在拆解這些物理。

---

# 📡 3.1 不同間距的單端過孔  
### 這一節在教你：
- 過孔之間距離越近 → **耦合越強**  
- 耦合越強 → **串擾（NEXT/FEXT）增加**  
- 串擾的來源：**磁場耦合 + 電場耦合**

### 對封裝的對應  
- micro‑bump pitch  
- TSV pitch  
- interposer via pitch  

封裝互連密度比 PCB 高 10 倍，這節的物理非常重要。

---

# 📡 3.2 不同半徑的單端過孔  
### 這一節在教你：
- via 半徑越大 → **電感變小**  
- via 半徑越小 → **電感變大 → IL/RL 變差**

### 對封裝的對應  
- TSV 直徑  
- micro‑via 直徑  
- ABF via 直徑  

封裝 via 的尺寸比 PCB 小很多，這節的直覺會直接用到。

---

# 📡 3.3 不同介電常數的單端過孔  
### 這一節在教你：
- 介電常數越大 → **寄生電容越大**  
- 寄生電容越大 → **阻抗下降 → 反射增加**

### 對封裝的對應  
- ABF Dk（3.0～4.0）  
- Si interposer Dk（11.9）  
- SiO₂ Dk（3.9）  

封裝材料比 PCB 更複雜，這節的物理會直接用到。

---

# 📡 3.4 不同反焊盤（anti-pad）大小  
### 這一節在教你：
- anti-pad 越大 → **寄生電容越小 → 阻抗上升**  
- anti-pad 越小 → **寄生電容越大 → 阻抗下降**

### 對封裝的對應  
- ABF via anti-pad  
- TSV landing pad  
- micro‑bump pad  

封裝設計常用 anti-pad 來調阻抗，這節非常重要。

---

# 📡 3.5 不同層數的單端過孔  
### 這一節在教你：
- via 穿越更多層 → **via barrel 更長 → 電感更大**  
- 電感越大 → **高頻損耗越大**

### 對封裝的對應  
- interposer via 深度  
- ABF via 深度  
- TSV 深度（最重要）  

TSV 的深度通常是 50–100 µm，這節的物理會直接用到。

---

# 📡 3.6 不同過孔長度  
### 這一節在教你：
- via 長度越長 → **電感越大 → IL/RL 變差**  
- via 長度越短 → **性能越好**

### 對封裝的對應  
- CoWoS interposer via  
- SoIC bonding via  
- ABF via  

封裝 via 的長度是 SI 的關鍵瓶頸之一。

---

# 📡 3.7 不同殘樁（stub）長度  
### 這一節是整章最重要的  
它在教你：

- stub = **未使用的 via 殘留部分**  
- stub 會形成 **λ/4 共振**  
- 造成嚴重反射（S11）與損耗（S21）  
- stub 越長 → 共振頻率越低 → 問題越嚴重

### 對封裝的對應  
- TSV stub（非常重要）  
- ABF via stub  
- micro‑via stub  

TSMC、Intel、AMD 都會用 **back-drill** 或 **via-in-pad** 來消除 stub。

這節的物理是封裝互連最重要的概念之一。
